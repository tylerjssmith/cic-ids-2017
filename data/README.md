# Download CIC-IDS2017 Benchmark Data
The Canadian Institute for Cybersecurity (CIC) has generated an intrusion detection evaluation dataset. The data can be obtained from [CIC](https://www.unb.ca/cic/datasets/ids-2017.html) and read by the project as follows (download links were current as of January 2026).

1. Visit [CIC](https://www.unb.ca/cic/datasets/ids-2017.html).
2. Click on "Download this dataset" at the bottom of the page.
3. Complete the form. After submission, the page will redirect.
4. Navigate to CIC-IDS-2017/CSVs/ and download MachineLearningCSV.zip.
5. Extract the files to data/raw/.
