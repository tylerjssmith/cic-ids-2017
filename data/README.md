# Download CIC-ID-2017 Benchmark Data
The Canadian Institute for Cybersecurity (CIC) has generated several benchmark datasets for network intrusion detection. These include CIC-IDS-2017, which contains network flows labeled as benign or one of several attack types. The data can be obtained from [CIC](https://www.unb.ca/cic/datasets/ids-2017.html) and read by the project as follows (download links were current as of January 2026).

1. Visit [CIC](https://www.unb.ca/cic/datasets/ids-2017.html).
2. Click on "Download this dataset" at the bottom of the page.
3. Complete the form. After submission, the page will redirect.
4. Navigate to `CIC-IDS-2017/CSVs/` and download `MachineLearningCSV.zip`.
5. Extract the CSV files to `data/raw/` (you will need to create `raw/`).
